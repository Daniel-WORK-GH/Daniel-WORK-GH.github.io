public project URL_Shortener(){
A simple but interesting project i made part of a python + sql learning process. Open a local web server using python, write a link to the textbox and get a shortened one.

The program will hash the link string and store it in the database, when someone enters the short link it will take him to the server and redirect to the desired site after fetchin it from the database.

If by some small chance the hash of the link already existing in the database, the program will find the next available one.

Github project repo : URL shortener files
}
